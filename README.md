# SMart-Boards
SmartHome Firmware for NodeMCU and ESP8266 devices


**MQTT Payloads:**

- Power topic payloads:
  - on
  - off
  - reboot
  
- Brightness topic payloads:
  - 1 to 255
  
- Effects topic payloads:
  - rainbow
  - rainbowwithglitters
  - confetti
  - bpm
  - juggle
  - sinelon
  - cyclonms
  - sparklems
  
- RGB topic payloads:
  - RGB values (R, G, B) --> (e.g.: (255, 0, 0) for Red color)
